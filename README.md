# SMART-HABIT-TRACKER
### Overview
Struggling to build consistent habits? Meet Habit Tracker - your personal companion for building lasting routines and tracking your productivity journey. This Java-based console application helps you establish, monitor, and analyze your daily habits with insightful productivity metrics.

Whether you're trying to read more, exercise regularly, or learn a new skill, Habit Tracker provides the structure and motivation you need to stay on track with your goals.

---
### Features
Core Functionality
Add New Habits - Create and customize your habit list
View All Habits - See your complete habit portfolio at a glance
Update Habits - Rename or modify existing habits
Remove Habits - Clean up habits you no longer track

---
### Progress Tracking
Daily Check-ins - Mark habits as completed each day
Productivity Scoring - Get daily completion percentages
Weekly Analytics - View 7-day performance averages
Trend Analysis - See if you're improving, declining, or staying stable

---
### Data Management
Automatic Saving - Your data persists between sessions
File-based Storage - Simple, portable data storage
Data Recovery - Pick up right where you left off

---

### Technologies Used
Java 8+ - Core programming language
Java Collections Framework - Data management (ArrayList)
Java Time API - Date handling and calculations
File I/O - Data persistence and storage
Object-Oriented Programming - Clean, maintainable architecture
