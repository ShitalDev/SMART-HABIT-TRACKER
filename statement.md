# Problem Statement
Many individuals struggle to maintain consistency in their daily habits due to a lack of structured tracking and awareness of their productivity trends. Traditional manual tracking methods are often unreliable and do not provide meaningful insights into performance over time. Therefore, there is a need for a simple, accessible, and persistent system that helps users manage habits and analyze their progress effectively.

# Scope of the Project
### The Habit Tracker with Productivity Analysis project includes:
A console-based interface for managing habits.
Adding, updating, viewing, and deleting habits.
Daily check-ins to mark habits as completed.
Persistent data storage using text files.
Calculation of daily productivity scores.
Weekly performance analysis.
Trend detection (Improving, Stable, Declining).
Modular Java implementation using object-oriented principles.

### The scope excludes:
UI/GUI development (e.g., JavaFX, web interface).
Cloud storage or database integration.
Notifications or reminders.
Advanced analytics or chart visualizations.
Target Users
Students who want to build consistent study habits.
Working professionals aiming to track personal routines.
Individuals looking to improve productivity and discipline.
Anyone who wants a simple, offline, no-installation habit tracker.

# High-Level Features
Habit Management: Create, view, edit, and delete habits.
Daily Check-Ins: Mark habits as completed each day.
Progress Tracking: View all check-ins for a date or for today.

# Productivity Analytics:
Daily Score
Weekly Score
Trend (Improving, Stable, Declining)
Persistent Storage: Saves habits and check-ins in text files for long-term use.
Menu-Driven Navigation: Easy-to-use console interface.
